# Stellar Payment dApp

## Project Description

A Stellar network decentralized application (dApp) built with Next.js and Tailwind CSS. This application allows users to seamlessly connect their Freighter wallet, view their account balances, and execute transactions on the Stellar Testnet.

## Features

- Connect with Freighter Wallet
- View Live Stellar Network Balance
- Send Testnet Transactions (XLM)
- Dedicated Transaction Status and Result Tracking

## Setup Instructions (How to run locally)

1. **Clone the repository** (if you haven't already):

   ```bash
   git clone <your-repo-url>
   cd stellar-payment-dapp
   ```

2. **Install dependencies**:

   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Run the development server**:

   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

4. **Open the App**:
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
